# Installation Instructions

## Quick Start

1. **Dependencies installieren:**
```bash
pip install -r requirements.txt
```

2. **GUI starten:**
```bash
python ultrastar_gui.py
```

3. **CLI verwenden:**
```bash
python ultrastar_generator.py song.mp3 --lyrics lyrics.txt
```

## System Requirements

- Python 3.8+
- FFmpeg (f端r Audio-Verarbeitung)
- Mindestens 4GB RAM
- Audio-Dateien: MP3, WAV, FLAC, M4A, OGG

## FFmpeg Installation

**Windows:**
- Download von https://ffmpeg.org/download.html
- Zur PATH Variable hinzuf端gen

**macOS:**
```bash
brew install ffmpeg
```

**Ubuntu/Debian:**
```bash
sudo apt update
sudo apt install ffmpeg
```

## Troubleshooting

**Problem: KivyMD Import Error**
```bash
pip install --upgrade kivymd kivy
```

**Problem: Audio Loading Error**
```bash
pip install --upgrade librosa soundfile
```

**Problem: Permission Error**
```bash
pip install --user -r requirements.txt
```
